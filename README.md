Online Shopping Store – Frontend Project

This project is part of the ProU Technology – Frontend Development Assessment.
It is a beginner-friendly, responsive online shopping web application built using HTML, CSS, JavaScript, and mock JSON data.

Users can browse categories, view products, search items, add items to the cart, view the total amount, and remove items.
The interface is clean, simple, colorful, and mobile-responsive.


Features

1. Category selection (Beauty, Kids Wear, Women Wear, Men Wear)
2. Product listing using mock JSON data
3. Search functionality
4. Add to cart and remove from cart
5. Total price calculation
6. Time-based greeting message (Good Morning, Good Afternoon, Good Evening)
7. Responsive layout
8. Simple and attractive design


1. Setup Steps

Follow the steps below to run this project:

Option 1: Open in Browser

1. Download or clone the repository.
2. Open the file *index.html* in any web browser.
3. The website will load immediately.

Option 2: Run with VS Code Live Server

1. Install Visual Studio Code.
2. Install the "Live Server" extension.
3. Right-click on *index.html* and select *Open with Live Server*.
